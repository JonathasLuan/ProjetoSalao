Passos:
1. Faça o download da biblioteca que você deseja adicionar. No caso do Bootstrap, você pode baixar os arquivos CSS e JS no site oficial (https://getbootstrap.com/docs/5.0/getting-started/download/).
2. Descompacte o arquivo que você baixou. Dentro da pasta, você encontrará arquivos CSS e JS que você precisará adicionar ao seu site.
3. Adicione os arquivos CSS e JS ao seu site. Você pode fazer isso de algumas maneiras:
* Copie os arquivos CSS e JS para as pastas correspondentes em seu site. Por exemplo, o arquivo CSS do Bootstrap geralmente é chamado "bootstrap.css" e o arquivo JS é chamado "bootstrap.js". Se o seu site tiver uma pasta chamada "css", copie o arquivo CSS do Bootstrap para essa pasta. Se o seu site tiver uma pasta chamada "js", copie o arquivo JS do Bootstrap para essa pasta. Certifique-se de que o caminho para esses arquivos seja correto.
* Se você estiver usando um sistema de gerenciamento de pacotes como o NPM ou o Yarn, pode instalar a biblioteca usando essas ferramentas. Por exemplo, você pode instalar o Bootstrap no NPM digitando "npm install bootstrap" em seu terminal. Isso irá baixar e instalar automaticamente os arquivos CSS e JS do Bootstrap em sua pasta node_modules.
* Se você estiver usando um sistema de compilação de front-end como o Webpack ou o Gulp, pode adicionar a biblioteca como uma dependência em seu arquivo de configuração. Isso permitirá que o sistema de compilação baixe e inclua automaticamente os arquivos CSS e JS da biblioteca em seu site.
4. Adicione os links para os arquivos CSS e JS em seu site. Para fazer isso, abra o arquivo HTML em que você deseja usar a biblioteca e adicione os seguintes links no cabeçalho:

<!-- Link para o arquivo CSS do Bootstrap -->
<link rel="stylesheet" href="caminho/para/bootstrap.css">

<!-- Link para o arquivo JS do Bootstrap (se necessário) -->
<script src="caminho/para/bootstrap.js"></script>
